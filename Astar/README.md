## 注意
这里有一部分代码是将实际坐标（单位：米）转换为了栅格坐标（i，j）。转换关系为x坐标转换为列，y坐标转换为行。在画图（plot）的时候要注意，画的是x，y坐标，需要把行和列再换回来，代码中栅格的行是数据的第一位，换成x后，就在数据的第二位了。
